# Consulting Case: EuroRetail Bank Profitability Analysis

**Python-gestützte Strategieanalyse für eine europäische Retail Bank**

## Überblick

Diese Case Study analysiert die Profitabilität einer fiktiven europäischen Retail Bank ("EuroRetail Bank") mit Python. Sie zeigt die Fähigkeit, strategische Business Cases mit datengetriebenen Methoden zu untermauern.

## Enthalten

- **Segmentanalyse**: Profitabilität nach Geschäftsbereichen (Retail, Corporate, Wealth Management, Investment Banking)
- **Regionalanalyse**: Umsatzverteilung über Regionen (Österreich, Deutschland, CE/SEE Europe)
- **Benchmarking**: Vergleich mit 4 Wettbewerbern (5 Kennzahlen)
- **What-If-Szenarien**: 3 strategische Optionen mit quantifizierten Auswirkungen
- **HTML-Report**: Professioneller Consulting-Bericht mit allen Charts

## Ausführen

```bash
pip install -r requirements.txt
python case_analysis.py
```
